## Overview

This project started as a homepage design created in Figma and was then translated into a functional front-end implementation. The goal was to preserve the original design intent while adapting it to real-world HTML and CSS constraints.

## What this project demonstrates

- Translating a visual design into working front-end code
- Maintaining layout structure and visual hierarchy from design to implementation
- Attention to spacing, alignment, and typography
- Understanding how design decisions affect technical execution

## Process

- Designed the homepage layout and structure in Figma
- Defined hierarchy, spacing, and key visual elements
- Implemented the design using HTML and CSS
- Adjusted details during development to ensure usability and responsiveness
